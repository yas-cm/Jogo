# Jogo de Detetive - Wolvesville

Este projeto é um jogo de lógica e dedução em modo texto.

## Como Rodar o Jogo

Para executar o projeto, siga os passos abaixo:

1. Clone esse repositório em sua máquina

2. Abra um terminal (cmd ou no vscode)

3. Navegue até onde está localizados os arquivos

4. Rode no terminal:

python jogo.py
