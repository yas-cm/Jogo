# Jogo de Detetive - Wolvesville

Este projeto é um jogo de lógica e dedução em modo texto.

## Como Rodar o Jogo

Para executar o projeto, siga os passos abaixo:

1.  Abra um terminal (Prompt de Comando, PowerShell, etc.).

2.  Navegue pelo terminal até o diretório onde este projeto está salvo e entre na pasta **`codigo`**.

3.  Uma vez _dentro_ da pasta `codigo`, execute o arquivo principal do jogo com o seguinte comando:

    ```bash
    python jogo.py
    ```

4.  O jogo será iniciado no seu terminal. Boa investigação!

Obs: Caso não consiga baixar a pasta "Códigos" no google collab, entrar no pistas.py e alterar o caminho do json de "codigos/pistas.json" para apenas "pistas.json"
